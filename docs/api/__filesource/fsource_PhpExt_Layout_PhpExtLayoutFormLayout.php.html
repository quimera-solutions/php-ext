  23.  *
  24.  *   - <b>hideLabels:</b> (Boolean) True to hide field labels by default (defaults to false)
  25.  *   - <b>itemCls:</b> (String) A CSS class to add to the div wrapper that contains each field label and field element (the default class is 'x-form-item' and itemCls will be added to that)
  26.  *   - <b>labelAlign:</b> (String) The default label alignment. The default value is empty string '' for left alignment, but specifying 'top' will align the labels above the fields.
  27.  *   - <b>labelPad:</b> (Number) The default padding in pixels for field labels (defaults to 5). labelPad only applies if labelWidth is also specified, otherwise it will be ignored.
  28.  *   - <b>labelWidth:</b> (Number) The default width in pixels of field labels (defaults to 100)

  32.  *   - <b>clearCls:</b> (String) The CSS class to apply to the special clearing div rendered directly after each form field wrapper (defaults to 'x-form-clear-left')
  33.  *   - <b>fieldLabel:</b> (String) The text to display as the label for this field (defaults to '')
  34.  *   - <b>hideLabel:</b> (Boolean) True to hide the label and separator for this field (defaults to false).

  36.  *   - <b>labelSeparator:</b> (String) The separator to display after the text of the label for this field (defaults to a colon ':' or the layout's value for labelSeparator). To hide the separator use empty string ''.
