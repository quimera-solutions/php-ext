PhpExt_Menu_Item

Description
Description | Descendents | Vars | Methods (details) Constants

A base class for all menu items that require menu-related functionality (like sub-menus) and are not static display items.

Item extends the base functionality of PhpExt_Menu_BaseItem by adding menu-specific activation and click handling.
